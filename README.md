# Clipboard Auto-Saver (Автоматический сборщик буфера обмена)

Легковесный консольный скрипт на Python для автоматического мониторинга системного буфера обмена и сохранения скопированного текста в файл в фоновом режиме.

Скрипт идеально подходит для быстрого сбора ссылок, контактов, адресов электронной почты, заметок и других текстовых данных без необходимости переключаться между окнами или вручную вставлять текст в файл.

---

## 🌟 Основные возможности

* **Полная автоматизация:** Скрипт отслеживает изменения буфера обмена в фоновом режиме. Достаточно скопировать текст любым способом (`Ctrl+C`, правой кнопкой мыши -> Копировать, или кнопкой на веб-странице).
* **Умное отслеживание:** Скрипт игнорирует дубликаты (не сохраняет один и тот же текст дважды подряд) и пустые строки.
* **Аудио-подтверждение:** При успешном сохранении воспроизводится короткий тихий звуковой щелчок (клик), информирующий о том, что данные записаны.
* **Безопасность данных (Сессии):** Для каждого запуска программы создается отдельный файл с временным штампом в названии (например, `Saved_2026-06-21_13-30-15.txt`), чтобы случайно не перезаписать старые данные.
* **Два режима работы:**
  1. **С разделителями и датой:** Каждая запись маркируется временем сохранения и отделяется пустой строкой.
  2. **Чистый текст:** Данные записываются строго друг за другом с новой строки без лишних символов.
* **Минимальное потребление ресурсов:** Проверка буфера происходит раз в секунду и потребляет **0% CPU**.

## 📋 Системные требования

* **Операционная система:** Windows (для работы звукового сигнала `winsound`).
* **Интерпретатор:** Python 3.6 или выше.
* **Библиотеки:** `pyperclip`.

## 🚀 Инструкция по использованию

### 1. Автосохранение буфера обмена (`clipboard_saver.py`)
1. Запустите скрипт через командную строку:
   ```bash
   python clipboard_saver.py
   ```
2. Скрипт предложит выбрать режим форматирования и настроить звук:
   * **Дата и разделители**: Нажмите `y` или **Enter** для сохранения с датой и временем. Введите `n` и нажмите **Enter** для сохранения только чистого текста (каждая запись с новой строки).
   * **Звуковой сигнал**: Нажмите `y` или **Enter**, чтобы оставить тихий звуковой щелчок-подтверждение. Введите `n` и нажмите **Enter** для полностью беззвучного режима работы.
3. Сверните окно консоли и продолжайте работу. Копируйте любой нужный текст. При успешном автосохранении в файл вы услышите тихий щелчок (если звук включен).
4. Результаты будут сохранены в файл вида `Saved_ГГГГ-ММ-ДД_ЧЧ-ММ-СС.txt` в папке со скриптом.
5. Для остановки мониторинга нажмите комбинацию клавиш `Ctrl + C` в окне консоли или просто закройте его.

---

### 2. Сортировка контактов (`contact_sorter.py`)
Если у вас есть файл со списком контактов в разном формате (например, собранный за несколько сессий), вы можете отсортировать его по категориям.

1. Создайте или скопируйте файл с именем **`Contacts.txt`** в папку со скриптом. Поместите туда все собранные контакты.
2. Запустите скрипт сортировки:
   ```bash
   python contact_sorter.py
   ```
3. Скрипт просканирует файл, очистит его от дубликатов и распределит контакты по отдельным файлам:
   * **`Telegram_(date)_(time).txt`** — для аккаунтов вида `@username` и ссылок `t.me/*`
   * **`Discord_(date)_(time).txt`** — для ссылок-инвайтов Discord
   * **`Mail_(date)_(time).txt`** — для e-mail адресов и неполных записей (типа `@gmail`, `@mail.ru`)
   * **`VK_(date)_(time).txt`** — для ссылок ВКонтакте (если они обнаружены)
   * **`Others_(date)_(time).txt`** — для прочих строк, которые не подошли под шаблоны выше.
4. В конце работы скрипт покажет подробный отчет со статистикой в консоли и сохранит файлы. Панель консоли закроется после нажатия клавиши **Enter**.

---

### 3. Сравнение с базой старых контактов (`contact_deduplicator.py`)
Этот скрипт позволяет исключить из нового файла `Contacts.txt` те контакты, которые вы уже собирали/обрабатывали ранее.

1. Убедитесь, что у вас есть старая база контактов под именем **`ContactsOLD.txt`** в папке со скриптом.
2. Поместите новые контакты в файл **`Contacts.txt`**.
3. Запустите скрипт сравнения:
   ```bash
   python contact_deduplicator.py
   ```
4. Скрипт проверит оба файла, удалит дубликаты из нового файла `Contacts.txt` (проверка идет без учета регистра, например, `@Username` и `@username` будут считаться дубликатами) и обновит его.
5. В консоли вы увидите отчет: сколько контактов было в старой базе, сколько добавлено новых, сколько совпадений удалено и сколько уникальных контактов осталось в `Contacts.txt`.
6. Окно консоли закроется после нажатия клавиши **Enter**.
