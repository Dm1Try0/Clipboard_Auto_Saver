# Clipboard Auto-Saver (Автоматический сборщик буфера обмена)

Легковесный консольный скрипт на Python для автоматического мониторинга системного буфера обмена и сохранения скопированного текста в файл в фоновом режиме.

Скрипт идеально подходит для быстрого сбора ссылок, контактов, адресов электронной почты, заметок и других текстовых данных без необходимости переключаться между окнами или вручную вставлять текст в файл.

---

## 🌟 Основные возможности

* **Полная автоматизация:** Скрипт отслеживает изменения буфера обмена в фоновом режиме. Достаточно скопировать текст любым способом (`Ctrl+C`, правой кнопкой мыши -> Копировать, или кнопкой на веб-странице).
* **Умное отслеживание:** Скрипт игнорирует дубликаты (не сохраняет один и тот же текст дважды подряд) и пустые строки.
* **Аудио-подтверждение:** При успешном сохранении воспроизводится короткий тихий звуковой щелчок (клик), информирующий о том, что данные записаны.
* **Безопасность данных (Сессии):** Для каждого запуска программы создается отдельный файл с временным штампом в названии (например, `Saved_2026-06-21_13-30-15.txt`), чтобы случайно не перезаписать старые данные.
* **Два режима работы:**
  1. **С разделителями и датой:** Каждая запись маркируется временем сохранения и отделяется пустой строкой.
  2. **Чистый текст:** Данные записываются строго друг за другом с новой строки без лишних символов.
* **Минимальное потребление ресурсов:** Проверка буфера происходит раз в секунду и потребляет **0% CPU**.

---

## 📋 Системные требования

* **Операционная система:** Windows (для работы звукового сигнала `winsound`).
* **Интерпретатор:** Python 3.6 или выше.
* **Библиотеки:** `pyperclip`.

## 🛠️ Установка и запуск

### 1. Клонирование репозитория
Откройте терминал/командную строку и склонируйте проект:
```bash
git clone https://github.com/ВАШ_ПРОФИЛЬ/TwitchTGParser.git
cd TwitchTGParser
```

### 2. Установка зависимостей
Установите библиотеку `pyperclip` для работы с буфером обмена:
```bash
pip install pyperclip
```

### 3. Запуск скрипта
Запустите скрипт через командную строку:
```bash
python clipboard_saver.py
```

## 🚀 Инструкция по использованию

1. После запуска скрипт предложит выбрать режим форматирования и настроить звук:
   ```text
   === НАСТРОЙКА АВТОСОХРАНЕНИЯ ===
   Добавлять дату и разделители к записям? (y/n, по умолчанию 'y'):
   Включить звуковой сигнал при сохранении? (y/n, по умолчанию 'y'):
   ```
   * **Дата и разделители**: Нажмите `y` или **Enter** для сохранения с датой и временем. Введите `n` и нажмите **Enter** для сохранения только чистого текста (каждая запись с новой строки).
   * **Звуковой сигнал**: Нажмите `y` или **Enter**, чтобы оставить тихий звуковой щелчок-подтверждение. Введите `n` и нажмите **Enter** для полностью беззвучного режима работы.
2. Сверните окно консоли и продолжайте работу.
3. Копируйте любой нужный текст. При успешном сохранении в файл (если включен звук) вы услышите тихий щелчок.
4. Результаты будут сохранены в файл вида `Saved_ГГГГ-ММ-ДД_ЧЧ-ММ-СС.txt` в папке со скриптом.
5. Для остановки мониторинга нажмите комбинацию клавиш `Ctrl + C` в окне консоли или просто закройте его.
